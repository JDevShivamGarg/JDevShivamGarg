## Featured Projects

### [Aegis — LLM Evaluation & Observability Framework](https://github.com/JDevShivamGarg/Aegis-LLM-Evaluator---Observability-Framework)
Dual-pathway LLM eval platform: local zero-dependency validator + async Celery-distributed telemetry client. Runs deterministic assertions, semantic similarity (all-MiniLM-L6-v2), safety scoring (unbiased-toxic-roberta), RAG grounding, and parallel multi-judge consensus pipelines with mean-score fallback. Cost aggregation to $10⁻⁶ USD, Plotly regression heatmaps, Slack/Discord threshold alerts.

`FastAPI` `PostgreSQL` `Redis` `Celery` `Streamlit` `Docker`

---

### [Lab Deployment Agent — Amdocs](https://github.com/JDevShivamGarg)
Python MCP server (~65 composable tools) automating the full Kubernetes lifecycle for Amdocs Openet Platform (5G/4G) and Charging stacks. Five-tier values merge, preflight CRD validation, async observability dashboard. Resolved VoltDB OOM, PVC corruption, and Gloo CRD ordering in production. Turned multi-day expert-dependent installs into single-operator deployments.

`Python` `MCP` `Kubernetes` `Helm` `Docker` `RHEL` `Elasticsearch` `Consul`

---

### [Dev-Deck](https://github.com/JDevShivamGarg/Dev-Deck)
Offline-first mobile app for spaced-repetition learning. SRS algorithm, three study modes (MCQ / Flashcard / Scenario), AI card generation via Groq (llama-3.3-70b-versatile), context-aware deduplication, streak tracking, JSON export.

`React Native` `Expo` `expo-sqlite` `Zustand` `Groq API` `TypeScript`

---

### [Source Sync — Narrative Analysis Tool](https://github.com/JDevShivamGarg/Source-Sync)
Multi-stage NLP pipeline: Universal Sentence Encoder embeddings → FAISS cosine-similarity search (0.5 threshold) → K-Means clustering → Gemini 1.5 Flash narrative synthesis. Outputs structured JSON reports categorizing narrative gaps by story type.

`Python` `Flask` `React 18` `TensorFlow Hub` `FAISS` `scikit-learn` `Gemini`

---

### [AirLens — Environmental Monitoring Platform](https://github.com/JDevShivamGarg/Smart-Environmental-Monitoring)
Real-time air quality and meteorological tracking across 34 Indian cities. FastAPI + APScheduler ETL pipeline, two-tier caching (99% API call reduction, 800ms → 50ms load time), SlowAPI rate limiting, Docker Compose.

`React 19` `FastAPI` `PostgreSQL` `APScheduler` `Docker Compose` `Nginx`
